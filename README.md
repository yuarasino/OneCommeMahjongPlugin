## Description

コメント中の麻雀牌っぽいテキストをイラストにしてくれるわんコメプラグイン

兼、わんコメでコメントを置き換えるようなプラグインの開発プロジェクトサンプル


## Requirements

- node.js 22.x
- pnpm 9.x


## Development

- rollup: commonjsのビルド
- biome: コード整形
- jest: ユニットテスト
